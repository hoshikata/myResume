對使用者的服務

為了幫助社群網站的使用者能更方便使用網站，做了一個輔助工具，而該工具受到了 3245 位使用者喜歡。

現在社群網站十分發達，我喜歡的社群網站可以讓使用者自行套用佈景，但因其必須懂 HTML、CSS 知識才能順利變更佈景，為了想讓其他使用者更好使用這功能，故在 2019 年時開始規劃如何製作一個網站工具，讓使用者不需要先備知識就能使用，為此便買了幾堂線上課程，將知識補充後在 2020 年 5 月把網站完成並公開。

興趣的開始

接觸網頁技術的一開始，便發現撰寫網頁是十分有趣且富有成就感的事，開啟了我對於網頁設計的興趣，接著國中時便開始設計個人網站，且因為喜歡各種有趣的版面故進行多次的改版，而後在自己待的補習班自告奮勇幫補習班架設官網。在大學時亦主動提出擔任系網的維修員，傾聽學系的需求來改變網頁，讓我慢慢將自己的知識轉變為解決問題的方法。

在記憶的殿堂挖出可以應用的知識

在就讀研究所期間，在數位學習的課程中各組必須做出一款數位桌遊，但在最關鍵數位技術這方面卻沒有方向，組員苦惱期間，我便提出了用 JavaScript 製作隨機呈現內容的網頁，讓專題的難關能順利度過，並且也協助有相似問題的他組，教如何使用 JavaScript 與網頁的使用，使最後能順利完成作品。

應用過往學過的知識或者以不同角度切入問題點，這點不只在上述例子出現，在曾經參加過的3D動畫比賽，亦將學過的技術轉個方向應用，讓動畫能簡單快速地完成。

未來發展

看到各個難題被解決，以及製作工具滿足使用者的需求，讓我由衷感到喜悅與成就感。而撰寫網頁也是個十分有趣的過程，常常 coding 到三更半夜還不願睡覺只為了釐清思緒完成某個功能。

前端的變化是十分快速，不能僅滿足於現在已經學會的知識，要時刻保持著學到底的態度才能站穩。而在這過程中磨練自己的觀念，並且更靈活的應用是我今後的課題。
